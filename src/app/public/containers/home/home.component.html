The assignment by EightCap

Create an application that has

  • An introduction page as a landing page, you can use https://lipsum.com/ to generate the paragraphs
  • Has a child page "about", you can use the same 3rd party service generate paragraphs
  • Make sure the child page is lazy loaded
  • Has a language selector which should be visible on every page and should have at least 3 languages available
